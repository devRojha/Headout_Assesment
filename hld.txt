Functional Requirements:

1. user can create events
2. user can create fetch all events
3. user can create fetch individual event by event id 
4. user can create delete and update event by event id 

5. user can create post of lost and find item
6. user can see the individual description of item
7. user can delete the post of lost and find item

8. A department can creat a Announcment
9. A department can delete a Announcment by id
9. A department can update a Announcment by id
10. user can feth all announcement
